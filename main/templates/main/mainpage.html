2주차 세션 정리

- 페이지 만들기

페이지 만드는 법

  1. main>template>main 폴더 내에 mainpage.html 파일 생성 후 작성
  2. views.py에 mainpage 함수 작성
  3. urls.py에 views.py의 mainpage 함수 연결
  4. python manage.py runserver - bash에서 실행!

새 페이지 만들어 연결하기

  1. 페이지 만드는 법을 참고해 새 페이지를 만든다
  2. html 파일에서 클릭할 항목에 href="{% url '-' %}" 을 추가해준다

template 언어

  1. views.py의 mainpage 함수 안에 template 작성
  2. 반복문/조건문을 html 내에서 사용 가능
  3. 겹치는 부분을 따로 분리하여 template 상속도 가능!

static

  1. project폴더 > static 폴더 > css, images 폴더
  2. css 폴더 안에 css 파일 생성하고 활용 가능
  3. image 폴더 안에 이미지 업로드하여 관리 및 사용
